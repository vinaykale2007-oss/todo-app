# 🚀 Space To-Do App

A futuristic space-themed To-Do application built using HTML, CSS, and JavaScript.  
This app helps users manage daily tasks with an interactive and visually appealing UI.

---

## ✨ Features

- ✅ Add and delete tasks
- ✔️ Mark tasks as completed
- 💾 Data stored using LocalStorage (persistent even after refresh)
- 🤖 AI-based task suggestions
- 📊 Real-time dashboard stats (total tasks)
- 🔊 Sound effects on actions
- 🌌 Animated space background (moving stars)
- 🎨 Futuristic glassmorphism UI design

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Animations + Glass UI)  
- JavaScript (DOM Manipulation, LocalStorage)

---

## 🚀 How to Run

1. Download or clone the repository  
2. Open folder in VS Code  
3. Open `index.html` in browser  
   or use Live Server
